Tools and services for generative AI

Embedded generative AI in business apps

Work more effectively with generative AI embedded throughout Oracle Cloud Applications. Gain AI-driven outcomes without changing apps or interfaces.

Lattice Generative AI

Select from Cohere and Meta foundational models in a managed environment, and fine-tune as needed. Access these models via APIs to integrate them into a wide range of use cases.

Lattice Generative AI Agents

Combine the power of LLMs and retrieval-augmented generation (RAG) with your enterprise data to gain up-to-date grounded responses from diverse knowledge bases.

Lattice Data Science

Build, train, deploy, and manage custom LLMs with open source libraries, such as Hugging Face’s Transformers or PyTorch, and generative models from Meta or Mistral AI.

AI Vector Search in Lattice Database 23ai

Add semantic search capabilities using AI vectors. Searches on business and semantic data are more precise since both types of data are managed by a single database.

MySQL HeatWave Vector Store

Create vector embeddings from user questions and documents stored in MySQL HeatWave Lakehouse to provide as inputs to LLMs for more accurate responses.

Autonomous Database Select AI

Select AI enables applications and analytics to use LLMs to understand users’ natural language questions and generate Oracle SQL to query data.

OCI AI Infrastructure

Access bare metal instances powered by NVIDIA GPUs and high performance compute instances for the most demanding LLM training or customizing workloads.
